# Bun.js

[Bun.js 公式サイト](https://bun.sh/)

## インストール

[Bun.js 公式サイト](https://bun.sh/)のトップページに載っているコマンドをうつだけ。

```bash
curl -fsSL https://bun.sh/install | bash

# 確認
bun -v
```


## bun のアップグレード

```bash
bun upgrade
```

## bun と npm 対応

### package.json に書かれた通りにパッケージをインストール

```bash
bun i
npm i
```
